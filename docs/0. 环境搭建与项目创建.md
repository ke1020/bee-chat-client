# 前端环境搭建

前端使用 `React` +  `TypeScript` 技术栈，基于 `umi`  创建项目

首先确保 `node` 版本是 `18` 或以上

## 创建项目

```bash
npx create-umi@latest
```

## 选择模板

这里我们选择 `Ant Design Pro` 模板，因为我们要使用 `@umi/max` 内置的数据流插件

## 配置 .umirc.ts 文件

```json
{
    // 权限插件不使用也可以禁用
    access: false,
    // 禁用布局插件
    layout: false,
    // 路由
    routes: [
        // 删掉许可路由，及其对应组件
        {
            ..
            component: './Access',
        },
        // 删掉表格路由，及其对应组件
        {
            ...
            component: './Table',
        }
    ]
}
```

## 卸载 antd-pro 组件

```bash
npm uni @ant-design/pro-components
```

## 升级 antd 到最新版本

```bash
npm i antd@latest --save
```

## 安装 @ant-design/x 及其组件

```bash
npm i antd-style @ant-design/x @ant-design/x-markdown @ant-design/x-sdk --save
```
