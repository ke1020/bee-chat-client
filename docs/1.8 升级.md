当前 @ant-design/x 组件已经升级到了 2.2.0。
我们的程序也来升级一下。
升级到 2.2.0 后，会有 Bubble.List 组件跑到左侧的 BUG。

我们可以使用 create。Styles 创建一个样式。
然后绑定到 Bubble.List 组件的 className 属性上。
官方后续可能会解决此问题。
大家暂时不要升级。
我们暂时这样处理。

```tsx
const useStyles = createStyles(({ css }) => {
    return {
        bubbleList: css`
            .ant-bubble-list-scroll-box {
                width: 100%;
                align-items: center;
            }

            .ant-bubble-list-scroll-content {
                max-width: 840px;
            }
        `,
    }
});

export default () => {
    return <Bubble.List
        className={styles.bubbleList}
    />
}
```
